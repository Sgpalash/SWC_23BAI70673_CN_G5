# 🌐 IP Addressing & Subnetting

A comprehensive collection of notes, examples, formulas, and practice problems covering the fundamentals of IP Addressing and Subnetting in Computer Networks.

## 📖 Overview

This repository is designed to help students, networking enthusiasts, and beginners understand how IP addressing and subnetting work. It includes theoretical concepts along with solved examples to strengthen practical understanding.

## 📚 Topics Covered

- Introduction to Computer Networks
- IPv4 Address Structure
- Binary and Decimal Conversion
- Classes of IP Addresses (A, B, C, D, E)
- Public vs Private IP Addresses
- Network ID & Host ID
- Subnet Masks
- CIDR (Classless Inter-Domain Routing)
- Variable Length Subnet Masking (VLSM)
- Fixed Length Subnet Masking (FLSM)
- Calculating Network Address
- Broadcast Address
- First & Last Usable Host
- Number of Hosts per Subnet
- Number of Subnets
- Supernetting
- Default Gateway
- Practice Questions
- Solved Numerical Examples

## 🛠️ Skills Learned

- IP Address Classification
- Binary Calculations
- Subnet Design
- Network Planning
- CIDR Notation
- VLSM & FLSM Implementation
- Address Calculation Techniques

## 🎯 Learning Outcomes

After completing this repository, you will be able to:

- Identify different classes of IPv4 addresses.
- Calculate Network and Broadcast addresses.
- Determine valid host ranges.
- Perform subnetting efficiently.
- Understand CIDR notation.
- Design networks using VLSM and FLSM.
